לאחר הקימפול של תוכנית בשפת **C** (במערכת לינוקס), מתקבל **קובץ הרצה (Executable)** בפורמט **ELF**.

---

## סוג הקובץ

**ELF – Executable and Linkable Format**

זהו הפורמט הסטנדרטי בלינוקס לקבצי הרצה, קבצי אובייקט וספריות משותפות.

---

## שם הקובץ

* אם מקמפלים כך:

```bash
gcc main.c
```

ייווצר קובץ בשם:

```
a.out
```

* אם מציינים שם מפורש:

```bash
gcc main.c -o my_program
```

ייווצר קובץ בשם:

```
my_program
```

---

## מאפיינים חשובים

* הקובץ הוא **קובץ בינארי**, לא קובץ טקסט
* מכיל:

  * קוד מכונה
  * מידע על קישור לספריות
  * נקודת כניסה (`main` / `_start`)
* ניתן להרצה ישירה:

```bash
./my_program
```

---

## משפט קצר למבחן

> לאחר קימפול תוכנית C מתקבל קובץ הרצה בינארי בפורמט ELF, אשר ברירת המחדל לשמו היא `a.out` אלא אם צוין שם אחר באמצעות הדגל `-o`.
