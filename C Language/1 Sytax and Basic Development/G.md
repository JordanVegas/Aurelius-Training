להלן **הסבר ברור ופשוט “לחופף”**, עם דוגמאות קצרות בשפת C.

---

## 1️⃣ מה הם קבצי Header (`.h`)?

**Header file** הוא קובץ שמכיל **הצהרות** (ולא מימושים), כדי שתוכניות אחרות ידעו:

* אילו פונקציות קיימות
* אילו טיפוסים / קבועים מוגדרים

בדרך כלל מכיל:

* הצהרות של פונקציות
* `#define`
* `typedef`
* `struct`

### דוגמה

**math_utils.h**

```c
int add(int a, int b);
```

**main.c**

```c
#include "math_utils.h"

int main() {
    int x = add(2, 3);
    return 0;
}
```

📌 הקובץ `.h` לא מריץ קוד – הוא רק “מודיע” לקומפיילר מה קיים.

---

## 2️⃣ מה הן פקודות קדם־מעבד (Preprocessor)?

ה־**Preprocessor** הוא שלב שרץ **לפני הקומפילציה**.

הוא מטפל בשורות שמתחילות ב־`#`.

דוגמאות לפקודות קדם־מעבד:

```c
#include
#define
#ifdef
#ifndef
#endif
```

👉 הוא **לא מבין C**, רק עושה החלפות טקסט.

---

## 3️⃣ `#include`

מכניס את תוכן הקובץ כאילו נכתב פיזית בקובץ הנוכחי.

```c
#include <stdio.h>   // ספרייה סטנדרטית
#include "myfile.h"  // קובץ מקומי
```

---

## 4️⃣ מה הם Macros (`#define`)?

**Macro** הוא כלל החלפה טקסטואלית.

### Macro פשוט (קבוע)

```c
#define PI 3.14
```

כל `PI` בקוד יוחלף ב־`3.14` לפני הקומפילציה.

---

### Macro עם פרמטרים

```c
#define SQUARE(x) ((x) * (x))
```

שימוש:

```c
int y = SQUARE(5);  // הופך ל: ((5)*(5))
```

📌 חשוב סוגריים – אחרת מתקבלים באגים.

---

## 5️⃣ למה משתמשים ב־Macros?

✔ קבועים
✔ קיצור קוד
✔ קוד תלוי־מערכת
✔ תנאים לקומפילציה

דוגמה:

```c
#define DEBUG 1

#if DEBUG
    printf("Debug mode\n");
#endif
```

---

## 6️⃣ Header Guards (הגנה מכפל include)

מונעים טעינה כפולה של אותו header.

```c
#ifndef MY_HEADER_H
#define MY_HEADER_H

int add(int a, int b);

#endif
```

---

## סיכום קצר “לחופף”

* **Header (.h)** – קובץ הצהרות, לא קוד רץ
* **Preprocessor** – שלב לפני הקומפילציה שעושה החלפות טקסט
* **Macros (`#define`)** – החלפות טקסט, לא פונקציות
* **`#include`** – מכניס קובץ לקוד
* **Header guards** – מונעים כפילויות

---

## משפט מוכן למבחן

> קבצי Header מכילים הצהרות ומוגדרים באמצעות פקודות קדם־מעבד. ה־Preprocessor רץ לפני הקומפילציה ומבצע החלפות טקסטואליות כגון `#include` ו־`#define`, כאשר Macros מאפשרים הגדרת קבועים או קוד כללי ללא בדיקת טיפוסים.
