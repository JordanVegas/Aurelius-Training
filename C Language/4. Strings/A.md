להלן **הסבר ברור ויסודי על מחרוזות (Strings) ב־C**: איך הן ממומשות, ואיך יודעים שהן נגמרות.

---

## מהי מחרוזת ב־C?

ב־C **אין טיפוס `string` מובנה**.
מחרוזת היא פשוט:

> **מערך של תווים (`char`) שמסתיים בתו מיוחד `'\0'`**

---

## איך מחרוזת ממומשת בזיכרון?

### דוגמה

```c
char str[] = "hello";
```

בזיכרון זה נראה כך:

| כתובת  | ערך  |
| ------ | ---- |
| str[0] | 'h'  |
| str[1] | 'e'  |
| str[2] | 'l'  |
| str[3] | 'l'  |
| str[4] | 'o'  |
| str[5] | '\0' |

📌 ה־`\0` (תו NUL) הוא **תו סיום המחרוזת**
📌 לכן האורך האמיתי הוא **6 תאים**, למרות שהמחרוזת מכילה 5 תווים

---

## איך יודעים שה־string נגמר?

🔹 ע״י בדיקה של התו `'\0'`

כל פונקציה שעובדת עם מחרוזות:

* קוראת תווים אחד־אחד
* **עוצרת כשהיא פוגשת `'\0'`**

### דוגמה

```c
int i = 0;
while (str[i] != '\0') {
    printf("%c\n", str[i]);
    i++;
}
```

---

## למה חייב להיות `'\0'`?

בלי `'\0'`:

* התוכנית תמשיך לקרוא זיכרון אקראי
* תגרום ל־**Undefined Behavior**
* באגים / קריסות

---

## דרכים להגדיר מחרוזות

### 1️⃣ אתחול אוטומטי (מומלץ)

```c
char s[] = "abc";
```

✔ הקומפיילר מוסיף `'\0'` לבד

---

### 2️⃣ אתחול ידני

```c
char s[] = {'a', 'b', 'c', '\0'};
```

---

### 3️⃣ מצביע למחרוזת קבועה

```c
char *s = "hello";
```

⚠️ אסור לשנות את התוכן:

```c
s[0] = 'H';  // ❌ Undefined Behavior
```

---

## פונקציות נפוצות למחרוזות (`<string.h>`)

* `strlen` – מחשבת אורך עד `'\0'`
* `strcpy` – מעתיקה מחרוזת
* `strcmp` – משווה מחרוזות
* `strcat` – מחברת מחרוזות

לדוגמה:

```c
printf("%lu\n", strlen("hello"));  // 5
```

---

## נקודות חשובות למבחן ⚠️

✔ מחרוזת היא `char[]`
✔ תמיד מסתיימת ב־`\0`
✔ האורך ≠ גודל המערך
✔ אין בדיקת גבולות אוטומטית
✔ בלי `'\0'` → Undefined Behavior
