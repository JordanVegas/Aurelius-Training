ההבדל בין `i++` לבין `++i` הוא **מתי מתבצע השינוי בערך של `i` ביחס לשימוש בביטוי**.

---

## `i++` – Post-increment (הגדלה אחרי השימוש)

* קודם משתמשים בערך הנוכחי של `i`
* **אחר כך** מעלים את `i` ב־1

```c
int i = 5;
int x = i++;
```

מה קורה?

* `x` מקבל `5`
* `i` הופך ל־`6`

---

## `++i` – Pre-increment (הגדלה לפני השימוש)

* קודם מעלים את `i` ב־1
* **אחר כך** משתמשים בערך החדש

```c
int i = 5;
int x = ++i;
```

מה קורה?

* `i` הופך ל־`6`
* `x` מקבל `6`

---

## דוגמה בהדפסה

```c
int i = 3;
printf("%d\n", i++);  // מדפיס 3
printf("%d\n", ++i);  // מדפיס 5
```

---

## מתי אין הבדל?

כאשר משתמשים רק בהגדלה, בלי להשתמש בערך בביטוי:

```c
i++;
++i;
```

בשני המקרים `i` יוגדל ב־1.

---

## טבלה מסכמת

| ביטוי | מתי מתבצעת ההגדלה | הערך שמוחזר |
| ----- | ----------------- | ----------- |
| `i++` | אחרי השימוש       | הערך הישן   |
| `++i` | לפני השימוש       | הערך החדש   |

---

## משפט קצר למבחן

> `i++` מחזיר את ערכו הנוכחי של `i` ורק לאחר מכן מגדיל אותו, בעוד `++i` מגדיל את `i` תחילה ומחזיר את הערך החדש.
