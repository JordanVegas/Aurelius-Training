## 1️⃣ מהו מערך (Array) ב-C?

מערך הוא **אוסף של איברים מאותו טיפוס**, המאוחסנים **ברצף רציף בזיכרון**.

לדוגמה: אוסף של מספרים שלמים, תווים וכו’.

---

## 2️⃣ הגדרת מערך

### תחביר כללי

```c
type name[size];
```

### דוגמה

```c
int arr[5];
```

או עם אתחול:

```c
int arr[5] = {1, 2, 3, 4, 5};
```

אפשר גם:

```c
int arr[] = {1, 2, 3};
```

הקומפיילר קובע את הגודל לבד.

---

## 3️⃣ מימוש בזיכרון

### עיקרון חשוב

* כל איברי המערך מאוחסנים **רציף בזיכרון**
* אין רווחים בין האיברים
* כל איבר תופס `sizeof(type)` בתים

### דוגמה

```c
int arr[4];
```

ב־32bit / 64bit:

* כל `int` = 4 בתים
* סה״כ: 16 בתים רציפים

```
כתובת:   1000  1004  1008  1012
ערך:    arr[0] arr[1] arr[2] arr[3]
```

📌 לכן ניתן לחשב כתובת:

```
&arr[i] = &arr[0] + i * sizeof(int)
```

---

## 4️⃣ גישה לאיברי מערך

### באמצעות אינדקס

```c
arr[0] = 10;
arr[1] = 20;

printf("%d\n", arr[0]);
```

* האינדקס מתחיל מ־0
* אין בדיקת חריגה מגבולות!

---

### באמצעות לולאה

```c
for (int i = 0; i < 5; i++) {
    printf("%d\n", arr[i]);
}
```

---

## 5️⃣ שם המערך

שם המערך:

* מצביע לכתובת האיבר הראשון
* **אינו משתנה רגיל**
* אי אפשר לבצע עליו השמה

```c
int arr[3];
arr = other;   // ❌ שגיאה
```

אבל:

```c
&arr[0] == arr   // ✔ נכון
```

---

## 6️⃣ מערך ו־sizeof

```c
int arr[10];
printf("%lu\n", sizeof(arr));      // 40
printf("%lu\n", sizeof(arr[0]));   // 4
```

מספר איברים:

```c
int n = sizeof(arr) / sizeof(arr[0]);
```

---

## 7️⃣ מערכים כפרמטרים לפונקציה

כאשר מעבירים מערך לפונקציה:

* מועברת הכתובת שלו (לא העתק)
* השינויים משפיעים על המערך המקורי

```c
void fill(int a[], int size) {
    for (int i = 0; i < size; i++) {
        a[i] = i;
    }
}
```

## 8️⃣ נקודות חשובות למבחנים

✔ מערכים רציפים בזיכרון
✔ אינדקס מתחיל מ־0
✔ אין בדיקת גבולות
✔ שם המערך הוא כתובת האיבר הראשון
✔ גודל המערך קבוע בקומפילציה (למערך סטטי)

---

## משפט סיכום למבחן

> מערך ב־C הוא מבנה נתונים המייצג אוסף איברים מאותו טיפוס המאוחסנים ברצף בזיכרון. הגישה לאיברי המערך מתבצעת באמצעות אינדקס, ושם המערך מייצג את כתובת האיבר הראשון.

אם תרצה – אוסיף:

* מערכים דו־ממדיים
* הקשר בין מערכים למצביעים
* שאלות טריק נפוצות במבחנים
